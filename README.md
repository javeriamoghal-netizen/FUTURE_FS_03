
# FUTURE_FS_03 - Local Business Website

## 📌 Project Overview
This project is a modern and responsive Local Business Website built for a salon named Taha Beauty Trends.
It includes a stylish UI and a booking section for customers to schedule appointments.

## ✨ Features
- Modern and attractive UI design
- Fully responsive (mobile & desktop)
- Service listing with pricing
- Appointment booking form
- Business information (contact, address, working hours)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
